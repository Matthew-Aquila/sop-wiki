# Design SOP System

## Overview
This site contains documentation for design workflows and asset identification processes for Aquila Fielding and GIS.

---

## Processes

### Basemap Workflow
Used to show exisiting conditions to be used by the Design team. *(Ex. [Edge of Pavement](basemap/EOP.md), [Centerlines](basemap/centerlines.md), Sidewalks, Right of Ways, ect.)*

- [Open Basemap Workflow](basemap/index.md)

---

### Asset Identification
Used to identify and classify assets from LiDAR. *(Ex. [Exisiting Power Poles](assets/poles.md), [Hand Holes](assets/ug.md), [Pedestals](assets/ug.md), ect.)*

- [Open Asset Identification](assets/index.md)

---

## Recommended Workflow Order

1. Basemap Workflow  
2. Asset Identification  

*Note: These may overlap depending on project needs.*
